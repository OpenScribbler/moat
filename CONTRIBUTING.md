# Contributing to MOAT

Feedback is welcome via [GitHub Issues](https://github.com/OpenScribbler/moat/issues).

Areas where input is especially valuable:

- Security considerations that are missing or under-specified
- Implementation experience (cross-language hashing behavior, Sigstore integration)
- Registry operator perspective on operational requirements
- Gaps in the spec or open issues

## Versioning and releases

For the versioning scheme, spec maturity stages, and release process, see [RELEASING.md](RELEASING.md).
